🧠 Why this beats the 6-ad-set version

NewsBreak has no CBO — budget lives on the ad set

✗ What the 6 ad sets did wrong

  • NewsBreak has no campaign budget optimization. Every ad set carries its own budget — the campaign can't move money between them.
  • So $150 ÷ 6 = ~$25 per ad set — and NB flagged $25 as too low to deliver.
  • At $25 each, no ad set collected enough conversions to leave the learning phase → partial delivery, no clear winner, budget wasted on under-trained sets.
  • Six sets also competed against each other in the same auction for overlapping people.

✓ What 1 ad set + 6 ads fixes

  • All $150/day in one ad set — comfortably above NB's minimum, with the volume to actually exit learning and optimize.
  • Drop all 6 distinct creatives inside it. NB rotates them, reads the response, and shifts spend to the winners automatically — the CBO-style optimization you wanted, done at the ad level (which is how NB actually works).
  • One auction entry, not six. No self-competition.
  • The creatives do the audience-splitting — doctor, wife, price, curiosity, stamina, search-pain — so one broad ad set still reaches many different buyers.
The core idea: on NewsBreak you don't spread budget to reach different people — you put the budget in one strong ad set and let different creatives reach different people. That's exactly what 6 unique ads in one set does.

Why the ad set is deliberately BROAD. With one ad set holding all the budget, you want a wide pool so NB's optimizer has room to match each of the 6 creatives to the right person. 🚀 How to launch it

  1. Fire the conversion pixel first. You're an affiliate, so you can't tag Rugiet's checkout — fire the NB pixel on a pre-lander/quiz you control, or run an Everflow → NewsBreak S2S postback. Optimize a mid-funnel event (quiz-start / qualified lead) to start — a raw purchase is too rare at $150/day to train on. Confirm one test conversion lands before spending.
  2. Create 1 campaign — objective Conversions.
  3. Create 1 ad set — budget $150/day, Location USA (exclude non-serviceable states), Gender Unlimited, Age 31–44 / 45–64 / 65+, Language English, account on PST, schedule 3 PM–12 AM PST, every day.
  4. Add all 6 ads to that one ad set. For each: upload its video + its image (9:16) as the poster, paste the headline as the title and the description as the body (insert {City}/{State} via + Add dynamic content), set CTA = "Learn More", Brand name = "Rugiet Ready", and upload the 40×40 logo.
  5. Launch all 6 at once so they compete from the same start and NB can pick a winner fairly. Don't touch it for 3–5 days — editing the budget or adding/removing ads resets learning.
  6. Then manage it: once it exits learning, pause the 1–2 clear losers, keep the winners, and swap in a fresh creative to replace a fatigued one. To scale, raise the ad-set budget in small ~20% steps (big jumps re-trigger learning).
