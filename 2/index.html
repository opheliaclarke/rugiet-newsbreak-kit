🕐 Best times to run — PST

USA-wide, so windows blanket all 4 US time zones

You're targeting all of the USA, so a PST window has to cover Eastern through Pacific. ED buys skew evening + private, and NewsBreak's older readers scroll early morning + night. Plan:

🌙 Primary — Evening
4:00–11:00 PM PST
Covers 7–11 PM in every US zone (ET evening = 4 PM PST → PT evening = 11 PM PST). Highest intent for ED — private, relevant, winding down. Put ~60% of budget here.
☀️ Secondary — Morning
4:00–8:00 AM PST
The older-male "coffee + news" scroll across ET→PT mornings. Strong for problem-aware / doctor / comparison ads. ~25% of budget.
🗓️ Days
All 7 · lean Thu–Sun
Weekends + Sunday night run hottest for men's-health impulse. Keep all days on; nudge budget to Thu–Sun evenings.
🧪 Learning phase
Days 1–4: all-day
Run 24/7 for the first 3–4 days so the algo gets conversion data, then clamp to the windows above. Day-parting too early starves learning.
Set it in your account's timezone. If your NewsBreak account is on PST, use the times above as-is. If it's on ET, shift +3 hrs (evening 7 PM–2 AM ET). The goal is the same: own the US-wide evening, catch the morning news scroll.

How to launch each ad

For every card: (1) download the image + the video, (2) in NewsBreak create the ad with both, (3) paste the headline as the title and the description as the body, (4) set targeting to the chips shown — Location USA, Gender, Age (highlighted buckets), Language English. Keep each campaign's ads together so the algo optimizes within one audience. Insert {City}/{State} via NewsBreak's + Add dynamic content in the title. Start 1 ad = 1 pairing, ~$100/day across the set, read by ~$800; kill losers by spend. Claims stick to Rugiet's real facts — 3-in-1 sublingual, ~15-min onset, up to 36 hrs, 81% prefer, doctor-prescribed. No fabricated FDA/cure claims (NB's Rx review kills those).
