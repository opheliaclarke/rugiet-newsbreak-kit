🚀 Rugiet × NewsBreak — 1 Campaign, 6 Ad Sets

One CBO campaign at $150/day → 6 ad sets → 1 ad in each (1 image + 1 video + 1 headline + 1 description). Each ad set is its own audience, so CBO can test all six and pour budget into the winners. Targeting is set at the ad-set level (that's where NewsBreak targets).

1 campaign · CBO $150/day total 6 ad sets · 1 ad each 1 img + 1 vid per ad

📐 Campaign structure

📣 1 Campaign — objective Conversions · CBO ON · budget $150/day
↳ 🎯 Ad set 1 — Older men · price & value (Male · 45-64, 65+) → 1 ad
↳ 🎯 Ad set 2 — Older men · doctor authority (Male · 45-64, 65+) → 1 ad
↳ 🎯 Ad set 3 — Broad men · curiosity hook (Male · 31-44, 45-64) → 1 ad
↳ 🎯 Ad set 4 — Men · Viagra switchers (Male · 45-64, 65+) → 1 ad
↳ 🎯 Ad set 5 — Women · wife testimonial (Female · 31-44, 45-64) → 1 ad
↳ 🎯 Ad set 6 — Women · partner reaction (Female · 31-44, 45-64) → 1 ad
What CBO does: you set the $150/day on the campaign only — never per ad set. NewsBreak spreads it across the 6 ad sets and automatically shifts money to whichever ones convert. The ~$25/day* on each card is just the even starting split ($150 ÷ 6) — CBO won't hold it there, it'll feed the winners. Don't set budgets on the ad sets themselves.

🕐 When to run — the simple answer

✅ Run 24/7 (all day, every day)

NewsBreak schedules on one clock — your account's timezone. Your buyers are all over the USA, so no single schedule can hit everyone's evening. So don't set a schedule at all — leave it running 24 hours. With CBO on, the campaign already spends more at the hours that convert, automatically. That beats any time window you'd guess.

🌍 Which timezone to pick?

It doesn't matter when you run 24/7 — leave the account default. Only if you later decide to limit hours: set the account to Eastern Time and run 6 PM–1 AM. Most Americans live in the Eastern/Central zones, so that covers prime evening for the majority — but do that only after ~4–5 days of data, not at launch.

How to launch

Build it like the map: 1 campaign (objective Conversions, CBO on, $150/day, schedule 24/7) → 6 ad sets, each with the targeting chips shown (Location USA · Gender · Age buckets · Language English) → 1 ad in each: download that card's image + video, paste the headline as the title and the description as the body. Insert {City}/{State} via + Add dynamic content in the title. Let it learn 3–5 days before judging; only pause an ad set after it's had real spend. Claims stick to Rugiet's real facts — 3-in-1 sublingual, ~15-min onset, up to 36 hrs, 81% prefer, doctor-prescribed. No fabricated FDA/cure claims (NB's Rx review kills those).
